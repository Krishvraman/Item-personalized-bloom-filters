To Conduct 100 trial experiment :
Please run 

python Main_Experiment.py 

The code will run 100 experiments

There are 10 sub -experiments  inside which 10 ratios of m/n are performed 

Winner for each experiment is selected - winner is either IBF(proposed) or SBF or neither




To get visual results :

Please run

python Testing_Auto.py



To manually control m, n and k of the standard and Item personalized bloom filters please run

python Testing.py 

inside the python script m and n and k are provided as variables
